# Client Image Assets

Drop files here to update the site. No code changes required — just replace the file and redeploy.

## Files

| Filename | Where It Appears | Recommended Format |
|---|---|---|
| `logo.png` | Navigation bar (top of every page) | PNG with transparent background, any size |
| `photo-1.jpg` | "Who We Are" team strip on homepage | JPG, landscape, min 800x500px |
| `photo-2.jpg` | Services page header (subtle background) | JPG, landscape, min 1200x600px |

## How to swap

1. Save your image with the exact filename above (e.g. `logo.png`)
2. Drop it in this folder, replacing the placeholder
3. Redeploy to Cloudflare Pages (or run `npm run dev` to preview locally)

## Tips

- **Logo:** Transparent background PNGs look cleanest on the dark nav
- **Photo 1:** Team photo, crew at a job site, or the owner — something personal
- **Photo 2:** Exterior of business, a truck/van, or equipment in use — works best with contrast since it's used as a faint background

If a file is missing or broken, that slot simply won't show — the layout adjusts automatically.
